# 🎨 Figma Color CLI

Convert Figma HEX colors into Flutter AppColor constants instantly.

## Install

```bash
dart pub global activate figma_color_cli
```

## Usage

```bash
figma-color "#254C82 #FFFFFF"
```

## Features

- Generate app_color.dart automatically
- Append / Replace existing colors
- Duplicate prevention
- HEX opacity support
